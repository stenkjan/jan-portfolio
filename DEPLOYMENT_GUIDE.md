## 🚀 Deployment Instructions

### 1. Vercel Deployment (Recommended)

```bash
# Connect to Vercel
npx vercel

# Follow prompts to deploy
# Project will be available at: https://jan-stenk-portfolio.vercel.app
```

### 2. Manual Deployment

```bash
# Build the project
npm run build

# Start production server
npm run start
```

### 3. Environment Variables

No environment variables required for basic functionality. For production contact form, add:

- Email service configuration
- Calendar API integration
