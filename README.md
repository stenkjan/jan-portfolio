# Jan Stenk - Developer Portfolio

A modern, responsive portfolio website showcasing my work as a React web app developer, Next.js specialist, Flutter developer, and full-stack TypeScript expert.

## 🚀 Features

- **Modern Tech Stack**: Built with Next.js 15, TypeScript, and Tailwind CSS
- **Responsive Design**: Mobile-first approach with seamless desktop experience
- **SEO Optimized**: Comprehensive metadata, structured data, and sitemap
- **Performance Focused**: Optimized images, lazy loading, and fast page loads
- **Interactive UI**: Smooth animations and modern user interface
- **Contact Integration**: Professional contact form with meeting scheduling

## 🛠️ Technologies Used

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React, Radix UI Icons
- **Deployment**: Vercel
- **SEO**: Structured data, sitemap, robots.txt

## 📂 Project Structure

```
src/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # Root layout with metadata
│   ├── page.tsx           # Homepage
│   ├── sitemap.ts         # SEO sitemap
│   └── robots.ts          # SEO robots.txt
├── components/
│   ├── ui/                # Reusable UI components
│   │   ├── Header.tsx     # Navigation header
│   │   └── StructuredData.tsx # SEO structured data
│   └── sections/          # Page sections
│       ├── HeroSection.tsx      # Hero/landing section
│       ├── AboutSection.tsx     # About me section
│       ├── ProjectsSection.tsx  # Projects showcase
│       └── ContactSection.tsx   # Contact form
└── types/                 # TypeScript definitions
```

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## 📈 SEO & Performance

- **Lighthouse Score**: Optimized for 90+ scores across all metrics
- **Core Web Vitals**: Fast loading and smooth interactions
- **Structured Data**: Rich snippets for search engines
- **Meta Tags**: Comprehensive social media and search optimization
- **Sitemap**: Auto-generated XML sitemap for search engines
